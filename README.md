This project focuses on predicting deforestation (forest loss area) using a machine learning approach called Support Vector Regression (SVR). A dataset was created with environmental and socio-economic factors such as CO₂ emissions, rainfall, population, GDP, illegal logging, and policy strictness. The data was preprocessed by handling missing values, encoding, and scaling, followed by training an SVR model. Initially, a linear kernel was used, and later optimized using GridSearchCV with both linear and RBF (non-linear) kernels to improve performance. The model was evaluated using metrics like MAE, MSE, RMSE, and R² score. Feature importance analysis revealed that CO₂ emissions, illegal logging, and GDP are the most influential factors affecting deforestation. Overall, the project helps in understanding key drivers of forest loss and provides insights that can support environmental decision-making and policy planning.
